💻 SOURCE CODE FILES & STRUCTURE
═══════════════════════════════════════════════════════════════════════════════

📁 include/ (Header files)
  ├─ game_console.h   - Console version declarations
  │                     Defines: GameState, Snake, Food, Position
  │                     Functions: InitGame, UpdateGame, DrawGame, etc.
  │
  └─ game.h           - Raylib version declarations  
                        (Same structures, for graphics version)

📁 src/ (Source code)
  ├─ game_console.c   - Console game logic (305 lines)
  │                     Core logic: collision, movement, rendering
  │
  ├─ main_console.c   - Console entry point (52 lines)
  │                     Main game loop, platform-specific code
  │
  ├─ game.c           - Raylib game logic (350+ lines)
  │                     Graphics rendering, Raylib integration
  │
  └─ main.c           - Raylib entry point (40 lines)
                        Raylib window setup, input handling

📁 build/ (Compiled executables)
  └─ snake_console.exe - Ready-to-play game (Windows)
                         Also: snake_console (Linux/macOS)

🔨 BUILD SCRIPTS
  ├─ build-console.bat  - Windows batch build (recommended)
  ├─ build-raylib.bat   - Optional Raylib build
  └─ Makefile          - Unix/Linux build
